Free Games Bot

This is a bot that i made together with my friend, because we thought it was a great idea.





Invite!


Prefix:

FG


Description:

This bot will be able to get the games that are currently free on the Epic Games Store with a custom API.


Commands:

- FG help to get the help menu.

- FG get to get all the games that are currently free.

- FG setup type this command to assign the notifications channels to the bot

- FG stop type this command to stop the bot sending notifications.
