Daily Art Jams

The Daily Art Jams bot was a fun and cool project to work on. It was the second bot i have made.





Invite!


Description:

This bot will send a random theme every 24 hour.

It will send a theme out of a list with more than 100+ themes.


I made it for a friend with a Art channel on Discord. I published the bot and can be used by everyone.


Commands:

- Start the bot with TB start (only for users with admin permissions).

- Stop the bot with TB stop (only for users with admin permissions).
